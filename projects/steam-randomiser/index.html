Steam Randomiser

Published on: 2021-04-15

A tiny CLI app built in Rust.

Picks one of the games installed on your Steam on your computer, and automatically launches it.

Installation

Cargo

You can easily install with cargo

cargo install steam_randomiser

Binary

You can download the latest binary release : https://github.com/ShadowMitia/steam_randomiser/releases
